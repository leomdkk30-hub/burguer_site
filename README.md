# burguer_site

Pequeno projeto de site para uma hamburgueria, criado como exercício do curso.

## Descrição

Site estático simples que demonstra a estrutura básica de uma página para uma hamburgueria: apresentação, menu, e contato. O objetivo é praticar HTML, CSS e JavaScript em um projeto leve.

## Recursos

- Layout responsivo básico
- Seção de menu (itens estáticos)
- Interações simples via JavaScript

## Estrutura do projeto

- `index.html` — página principal
- `css/style.css` — estilos do site
- `js/script.js` — scripts JavaScript
- `README.md` — este arquivo

## Como usar

1. Abra o arquivo `index.html` no seu navegador (duplo clique ou via `File → Open`).
2. Para desenvolvimento, use um servidor local (recomendado) — por exemplo, com Python:

```bash
python3 -m http.server 8000
# e então abra http://localhost:8000
```

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
